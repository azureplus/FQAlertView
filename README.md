# 安装
## CocoaPods

1. 在 Podfile 中添加 pod 'FQAlertView'。
2. 执行 pod install 或 pod update。
3. 导入 <FQAlertView/FQAlertView.h>。

## 手动安装
1. 下载项目demo文件。
2. 添加FQAlertViewDemo/FQAlertView文件夹中所有文件到项目中。
3. 导入 "FQAlertView.h"。
